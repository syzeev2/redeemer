Topic: How to use the tool 
Dear user, 

Basic running procedure -> Open input folder and put promos in promos.txt , tokens in tokens.txt , vccs in vccs.txt 

Customizations -> open config.json and put the address informations and the custom branding information 

Proxy Usage -> open proxies.txt and put rotating proxy none other than that otherwise it wont work.

Formats -> vcc format = ccn:vccexpyearexpmonth:cvv , token format -> token only or email:pass:token or pass:email:token none other than that
